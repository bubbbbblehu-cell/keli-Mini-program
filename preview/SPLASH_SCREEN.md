# 开屏页配置说明

## 📸 开屏图片

开屏页已集成，需要添加开屏图片文件：

### 文件位置

在 `preview/` 目录下放置以下文件：
- `splash.png` - 开屏图片（推荐尺寸：414x896 或更高分辨率）

### 图片要求

- **格式**：PNG（推荐）或 JPG
- **尺寸**：建议 414x896（iPhone标准尺寸）或更高
- **内容**：应用启动画面、Logo、品牌元素等
- **大小**：建议小于 500KB，确保快速加载

### 如果没有图片文件

如果 `splash.png` 不存在，开屏页会显示渐变背景色（蓝紫色渐变），不会报错。

## 🎨 自定义开屏页

### 修改显示时间

在 `preview/index.html` 中找到：

```javascript
setTimeout(() => {
    // 开屏页消失逻辑
}, 2500); // 修改这个数字（毫秒）来调整显示时间
```

### 修改淡出动画时间

找到：

```javascript
splash.style.opacity = '0';
setTimeout(() => {
    splash.remove();
}, 800); // 修改这个数字（毫秒）来调整淡出时间
```

### 修改背景色

如果图片加载失败，会显示渐变背景。可以在CSS中修改：

```css
#splash-screen {
    background: white; /* 或改为其他颜色/渐变 */
}
```

## 📱 移动端优化

开屏页已配置移动端优化：

- ✅ 全屏显示
- ✅ 防止滚动
- ✅ 白色背景（防止黑色闪现）
- ✅ 平滑淡出动画
- ✅ Apple Web App 支持

## 🔍 测试

1. **有图片时**：
   - 打开页面会显示 `splash.png`
   - 2.5秒后开始淡出
   - 0.8秒后完全消失

2. **无图片时**：
   - 显示渐变背景
   - 同样2.5秒后淡出

## 💡 提示

- 开屏图片建议使用应用Logo或品牌元素
- 保持图片简洁，避免过多文字
- 确保图片在不同设备上都能正常显示
